# ***时间复杂度分析***
## **添加操作(增) => O(n)**
### addLast(e) => O(n)
### addFirst(e) => O(1)
### add(index) => O(n/2) = O(n)
---
## **删除操作(删) => O(n)**
### removeLast(e) => O(n)
### removeFirst(e) => O(1)
### remove(index,e) => O(n/2) = O(n)
---
## **修改操作(改) => O(n)**
### set(index,e) => O(n)
---
## **查找操作(查) => O(n)**
### get(index) => O(n)
### contains(e) => O(n)
